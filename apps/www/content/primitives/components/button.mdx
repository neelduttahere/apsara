## Installation

Install the component from your command line.

```bash
npm install @odpf/apsara
```


## Anatomy
Import all parts and piece them together.

```tsx outline=true
import { Button } from '@odpf/apsara'

<Button>Basic button</Button>
<Button variant="primary" size="small">Basic button</Button>
<Button variant="primary">Basic button</Button>
<Button variant="primary" size="large">Basic button</Button>
<Button variant="primary" size="circle">B</Button>
<Button variant="secondary">Basic button</Button>
<Button variant="secondary" size="large">Basic button</Button>
<Button variant="secondary" outline>Basic button</Button>
```
