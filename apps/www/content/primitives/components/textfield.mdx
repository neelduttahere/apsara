## Installation

Install the component from your command line.

```bash
npm install @odpf/apsara
```


## Anatomy
Import all parts and piece them together.

```tsx outline=true
import { TextField } from '@odpf/apsara'


<TextField
  type="email"
  size="2"
  placeholder="Email"
  autoComplete="off"
/>
<TextField
  type="password"
  size="2"
  placeholder="Password"
  autoComplete="off"
/>
```
